# Dominick's Price Elasticity IV-DML

Dominick's retail scanner result tables are summarized into a compact portfolio project.  
The project estimates own-price sales responses across cereal, canned soup, bottled juices, and cookies using FE-IV, dynamic FE-IV, and FD-PLIV-DML result tables.

## Project Question

Can cost-driven price variation be used to estimate retail price elasticity after controlling for product-store and week fixed effects?

## Repository Structure

```text
notebooks/

  02_fe_iv_baseline_results.ipynb

  figures/                # Figures generated by notebooks
data/
  README.md               # Raw data note
```

## Notebook Order

1. `01_results_overview.ipynb` checks the available result tables and category coverage.
2. `02_fe_iv_baseline_results.ipynb` summarizes baseline FE-IV estimates and first-stage strength.
3. `03_dynamic_iv_robustness.ipynb` compares current-price and distributed-lag IV estimates.
4. `04_dml_validation_interpretation.ipynb` compares FE-IV with FD-PLIV-DML own-sales estimates.

## Main Takeaway

The estimates generally support a negative own-price sales response, but the magnitude is sensitive to controls, lag structure, and estimation strategy. The portfolio interpretation therefore emphasizes robustness and identification checks rather than a single elasticity number.
